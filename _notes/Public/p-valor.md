# Valor P

EL valor p es el que representa el error porcentual que tiene el rechazar el $H_0$

$$V_p = P(Z > Z_c)$$

Donde $Z_c$ es el zeta critico que utilizamos para juzgar a la hipotesis nula dentro de una [[pruebas de hipotesis]].
